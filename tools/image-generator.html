WYQ 博客 — 图片资源批量生成器

在浏览器里把 SVG 转成 PNG，一键下载全部 OG 图 + PWA 图标。
生成完后把 ZIP 解压到仓库根目录的 assets/og/ 和 assets/icons/ 即可。

1. 文章 OG 图 (1200×630)

用于微信、Twitter、Telegram 等社交分享卡片。

2. PWA / Apple Touch 图标

192×192 / 512×512 (PWA) + 180×180 (apple-touch-icon)。

3. 用法说明

  1. 点上方"下载所有...PNG" — 浏览器会逐个保存为 post-XXX.png 等。
  2. 把 OG PNG 放到仓库的 assets/og/ (与现有 SVG 同名)。
  3. 把图标 PNG 放到仓库的 assets/icons/。
  4. 提交 + push，微信/Twitter 分享时会自动生效。

提示：站点已经把 SVG 配为 og:image 的兜底；许多平台 (Twitter, Discord, Telegram) 直接支持 SVG。但微信、LinkedIn 较保守，建议生成 PNG 上传。
